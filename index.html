My Projects

Data Analysis Python and Tableau Project

This project analyzes an auto insurance claims dataset to uncover insights about potential fraudulent claims and how claim amounts vary across different customer segments. Using Python with an Isolation Forest model, the project flags around 92 claims as potentially fraudulent, while Tableau dashboards provide interactive visualizations that segment claim amounts by demographics such as gender, marital status, and education. The integration of machine learning and visual analytics helps improve risk management and tailor insurance strategies.

Pet Store Database Development

As part of a team, I contributed to the design and implementation of a MySQL database system for a pet store chain, optimizing the management of inventory, sales, employees, and customer data. I played a key role in developing the Entity-Relationship Diagram (ERD) and defining business rules to maintain data accuracy and consistency. Additionally, I created SQL scripts for database setup, data population, and advanced queries to analyze sales trends, employee productivity, and customer behaviors using Azure Data Studio. Throughout the project, I collaborated with team members to gather requirements, tackle technical challenges, and deliver actionable insights.

Credit Risk Analysis

This project analyzed credit card default risk using a UCI dataset to identify factors influencing default probability and improve credit risk models.The analysis also revealed that higher credit utilization and poor payment history were strongly linked to increased default rates. These findings can help financial institutions enhance credit scoring and risk management strategies.
